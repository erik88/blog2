--- metaTitle: Setting up C# Omnisharp for VS Code date: "2023-04-21" ---
  1. First make sure that you have installed:
    • The dotnet sdk
    • vscode
    • The omnisharp plugin:
  2. Restart VS Code
  3. We need to figure out where dotnet is installed. Open a terminal and type:
    Linux/Mac
    which dotnet
    Windows Powershell
    $(Get-Command dotnet).Path
  4. Copy the entire path, excluding the file itself.
  5. Open the VS Code settings. They are typically available under File->Preferences or Edit->Preferences. Or, use the hotkey Ctrl+P, then type >settings.
  6. Search the settings for dotnet path
  7. Paste the copied path.
